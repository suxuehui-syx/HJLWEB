《Web应用基础》课程结业报告

一、实验内容

本次实验做了一个有关零食推荐的静态网站，网站主要有11个网页，其中有四个子页面内容风格大体相同。网页主要由文字，图片构成。本次实验使用了CSS框架中的MDB。

二、开发过程

首先通过棋歌网和b站视频学习了HTML和CSS的相关知识，然后通过棋歌网对MDB的讲解以及自行上网查阅资料对MDB有了进一步的学习。实验环境我用的VSCode，首先建立网页的主页index.html文件，由主页通过导航添加超链接的方式跳转到其他不同页面去，每个子页面的排版我都使用了同主页面相同的方式，都包含了一个相同的导航条，这样更方便使用者在不同页面间的跳转。本次实验采用的是响应式的界面，可以方便不同屏幕尺寸的使用者获得更好的体验。

三、遇到的问题

排版布局有些混乱，展示图片的时候，图片大小尺寸放入网页中并不合适，导致在网页上面，文案和图片非常混乱，很难看
GitHub托管时git push没有权限的问题，已设置公钥，想要git push的时候，总是失败
配置SSH keys时，没有在指定路径产生.ssh文件夹，无法复制id_rsa.pub文件里的内容

四、解决方法

利用图片编辑软件修改图片尺寸或更改背景图片位置；三种定位方法：1）关键字：background-position: top left;2）像素：background-position: 0px 0px;3）百分比：background-position: 0% 0%;
看本地的.git/config设置的仓库url地址和github使用的链接地址是否一致如下图,如use https,则url需要用https的仓库地址
没有.ssh文件解决办法是: $ ssh-keygen -t rsa -C “邮箱地址” ##回车 Generating public/private rsa key pair. Enter file in which to save the key (/c/Users/Administrator/.ssh/id_rsa): ##回车 Created directory ‘/c/Users/Administrator/.ssh’. Enter passphrase (empty for no passphrase): ##回车 Enter same passphrase again: ##回车

五、总结

通过这段时间的学习，我基本了解了web开发的相关知识。但实践自己做起来却发现许多的问题，需要自己去查询资料解决问题，自己运用知识做实验时根本不熟系也记不住，很多地方都需要反复去翻阅资料。本次实验制作的网页比较简单，也不是很好看，但是第一次尝试自己做网页还是很开心，虽然过程中遇到了一些困难，但现在网络上有很多学习资料，通过查阅资料，就能够解决大部分问题。通过本次实验，我对web设计的知识有了更加深刻的认识，尽管自身依然存在着很多不足，但是网页的制作让我感到比单纯的编程更加快乐有趣，希望通过后期的努力，我能制作更加漂亮实用的网页。
